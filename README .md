# Blog
##React + Express + MongoDB

react全家桶+express实战个人博客系列教程

## 功能描述

### 前端部分
--------------------
* 文章列表展示
* 文章分类
* 登录管理权限管理
* 文章详情页展示
* 管理员文章管理
* 管理员标签管理
* 发文（支持MarkDown语法）


### 后端部分
--------------------
* mongoose数据库操作
* 路由管理
* 身份验证
* 基本的增删改查
* ...

### 技术栈
--------------------
* react
* react-redux
* react-router
* redux-saga
* babel
* webpack
* Express
* Mongodb
* Mongoose

该项目是学习作者 Neal_yang 的博客项目
